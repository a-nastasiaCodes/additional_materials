#### 1. Основы работы с параметрами

Функции могут принимать данные через **параметры**. Параметры указываются в скобках при определении функции. При вызове функции параметрам передаются значения (**аргументы**).

**Пример:**
```python
def say_hello(name):          # name - это параметр
    print(f"Привет, {name}")

say_hello("Иван")              # "Иван" - это аргумент
say_hello("Пётр")
say_hello("Мария")
# Вывод:
# Привет, Иван
# Привет, Пётр
# Привет, Мария
```

*   **Передача по позиции:** Значения сопоставляются параметрам в том порядке, в котором они записаны при вызове.

```python
def print_person(name, age):
    print(f"Имя: {name}")
    print(f"Возраст: {age}")

print_person("Иван", 37)       # name = "Иван", age = 37
# Вывод:
# Имя: Иван
# Возраст: 37
```

#### 2. Значения параметров по умолчанию

Можно сделать параметры необязательными, присвоив им значение прямо при определении функции.

*   **Правило:** Необязательные параметры должны идти **после** обязательных.

```python
# Параметр age необязательный, по умолчанию равен 18
def print_person(name, age=18):
    print(f"Имя: {name}  Возраст: {age}")

print_person("Пётр")           # Используется age по умолчанию (18)
print_person("Иван", 37)       # Переданное значение 37 переопределяет умолчание
# Вывод:
# Имя: Пётр  Возраст: 18
# Имя: Иван  Возраст: 37
```

Можно сделать все параметры необязательными:
```python
def print_person(name="Иван", age=18):
    print(f"Имя: {name}  Возраст: {age}")

print_person()                 # Имя: Иван  Возраст: 18
print_person("Пётр")            # Имя: Пётр  Возраст: 18
print_person("Алексей", 37)     # Имя: Алексей  Возраст: 37
```

#### 3. Именованные аргументы и специальные параметры

При вызове функции можно явно указывать, какому параметру какое значение присвоить, используя **имена параметров**.

```python
def print_person(name, age):
    print(f"Имя: {name}  Возраст: {age}")

# Передача по имени (порядок не важен)
print_person(age=22, name="Иван")
```

Для гибкости настройки того, *как* можно передавать аргументы, используются символы `*` и `/`.

*   **`*` (звездочка):** Все параметры **справа** от `*` могут быть только **именованными** (keyword-only).
*   **`/` (слеш):** Все параметры **слева** от `/` могут быть только **позиционными** (positional-only).

```python
# Параметры age и company - только по имени
def print_person(name, *, age, company):
    print(f"Имя: {name}  Возраст: {age}  Компания: {company}")

print_person("Пётр", age=41, company="Яндекс")  # OK
# print_person("Пётр", 41, "Яндекс")            # Ошибка!

# Параметр name - только по позиции
def print_person(name, /, age, company="Сбер"):
    print(f"Имя: {name}  Возраст: {age}  Компания: {company}")

print_person("Иван", 24)                            # OK (name по позиции)
print_person("Иван", company="Тинькофф", age=24)   # OK
# print_person(name="Иван", age=24)                 # Ошибка для name!

# Комбинированный пример
def print_person(name, /, age=18, *, company):
    print(f"Имя: {name}  Возраст: {age}  Компания: {company}")

print_person("Алексей", company="VK")               # OK
print_person("Иван", 37, company="Лаборатория Касперского") # OK
print_person("Пётр", company="Ozon", age=42)        # OK
```

#### 4. Неопределенное количество параметров (`*args`)

Если нужно передать в функцию произвольное количество аргументов, используется параметр со звездочкой (`*`). Внутри функции он становится кортежем (tuple).

```python
def sum_numbers(*numbers):      # numbers - это кортеж из переданных чисел
    result = 0
    for n in numbers:
        result += n
    print(f"Сумма = {result}")

sum_numbers(1, 2, 3, 4, 5)      # Сумма = 15
sum_numbers(3, 4, 5, 6)         # Сумма = 18
```

#### 5. Возврат значения (`return`)

Оператор `return` используется для двух целей:
1.  Возврат результата работы функции.
2.  Немедленный выход из функции.

**Возврат результата:**
```python
def get_message():
    return "Привет, Дмитрий!"      # Возвращаем строку

message = get_message()
print(message)                  # Привет, Дмитрий!

def double(number):
    return 2 * number           # Возвращаем результат выражения

print(double(5))                # 10
```

**Досрочный выход из функции:**
Если `return` используется без значения или в функции, которая ничего не возвращает, он просто завершает её работу.
```python
def print_person(name, age):
    if age > 120 or age < 1:
        print("Недопустимый возраст")
        return                  # Выходим из функции, ничего не печатая дальше
    print(f"Имя: {name}  Возраст: {age}")

print_person("Иван", 22)          # Имя: Иван  Возраст: 22
print_person("Пётр", -102)        # Недопустимый возраст
```

### Сводная таблица: Как работает vs Как не работает

| Тема | ✅ Как работает (правильно) | ❌ Как не работает (ошибки/проблемы) |
| :--- | :--- | :--- |
| **Простые параметры** | `def greet(name): print(name)` <br> `greet("Ольга")` | `greet()` -> Ошибка, пропущен аргумент. |
| **Позиционность** | `def func(a, b): print(a, b)` <br> `func(1, 2)`  # a=1, b=2 | Попытка передать больше аргументов, чем параметров: <br> `func(1, 2, 3)` -> Ошибка. |
| **Значения по умолчанию** | `def func(a, b=10): print(a, b)` <br> `func(5)`    # a=5, b=10 <br> `func(5, 20)` # a=5, b=20 | `def func(a=10, b): ...` -> **Ошибка синтаксиса**. Необязательный параметр `a` не может стоять перед обязательным `b`. |
| **Именованные аргументы** | `def func(name, age): ...` <br> `func(age=25, name="Дмитрий")` | `def func(name, age): ...` <br> `func(name="Дмитрий", 25)` -> Ошибка. Позиционный аргумент (`25`) идет после именованного. |
| **Только позиционные (`/`)** | `def func(a, /, b): ...` <br> `func(1, b=2)` # a - позиционный | `def func(a, /, b): ...` <br> `func(a=1, b=2)` -> Ошибка. `a` передан по имени, хотя должен быть по позиции. |
| **Только именованные (`*`)** | `def func(*, a, b): ...` <br> `func(a=1, b=2)` | `def func(*, a, b): ...` <br> `func(1, 2)` -> Ошибка. Аргументы переданы по позиции, хотя должны быть именованными. |
| **Несколько аргументов (`*args`)** | `def sum(*nums): print(sum(nums))` <br> `sum(1, 2, 3)` # Работает | `def sum(*nums): ...` <br> Попытка добавить обычный параметр после `*nums`: <br> `def sum(*nums, extra): ...` -> `extra` можно будет передать только по имени. |
| **Return (возврат)** | `def add(a, b): return a + b` <br> `x = add(3, 4)` # x = 7 | `def bad_add(a, b):` <br> &nbsp;&nbsp;&nbsp;&nbsp;`result = a + b` <br> `x = bad_add(3, 4)` # x = None (забыли return) |
| **Return (выход)** | `def check(x):` <br> &nbsp;&nbsp;&nbsp;&nbsp;`if x < 0: return` <br> &nbsp;&nbsp;&nbsp;&nbsp;`print(x)` | Код после `return`, который никогда не выполнится: <br> `def foo():` <br> &nbsp;&nbsp;&nbsp;&nbsp;`return 5` <br> &nbsp;&nbsp;&nbsp;&nbsp;`print("Секрет")` # Бесполезная строка |
