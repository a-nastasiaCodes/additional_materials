# Методы строк в Python

| Название метода | Описание | Пример использования (ввод → вывод) |
|---|---|---|
| **center()** | Выравнивает строку по центру с заполнением указанным символом | `'алгоритм'.center(14, '-')` → `'---алгоритм---'` |
| **count()** | Возвращает количество вхождений подстроки в строке | `'ракушки на берегу ракушки'.count('ракушки')` → `2` |
| **find()** | Ищет подстроку и возвращает индекс первого вхождения (или -1) | `'алгоритм'.find('ритм')` → `4` |
| **rfind()** | Ищет подстроку и возвращает индекс последнего вхождения | `'ракушка ракушка'.rfind('раку')` → `9` |
| **index()** | Аналогично find(), но вызывает исключение при отсутствии | `'питон'.index('т')` → `3` |
| **rindex()** | Аналогично rfind(), но с исключением при отсутствии | `'питон'.rindex('и')` → `1` |
| **startswith()** | Проверяет, начинается ли строка с указанной подстроки | `'Python'.startswith('Py')` → `True` |
| **endswith()** | Проверяет, заканчивается ли строка указанной подстрокой | `'file.txt'.endswith('.txt')` → `True` |
| **swapcase()** | Меняет регистр символов на противоположный | `'PyThOn'.swapcase()` → `'pYtHoN'` |
| **capitalize()** | Переводит первый символ в верхний регистр, остальные в нижний | `'пРИВЕТ'.capitalize()` → `'Привет'` |
| **upper()** | Все символы в верхний регистр | `'hello'.upper()` → `'HELLO'` |
| **lower()** | Все символы в нижний регистр | `'HELLO'.lower()` → `'hello'` |
| **title()** | Первую букву каждого слова в верхний регистр | `'hello world'.title()` → `'Hello World'` |
| **strip()** | Удаляет указанные символы с обоих концов (пробелы по умолчанию) | `'  text  '.strip()` → `'text'` |
| **lstrip()** | Удаляет указанные символы слева | `'  text  '.lstrip()` → `'text  '` |
| **rstrip()** | Удаляет указанные символы справа | `'  text  '.rstrip()` → `'  text'` |
| **split()** | Разбивает строку на список по разделителю (пробел по умолчанию) | `'a,b,c'.split(',')` → `['a', 'b', 'c']` |
| **rsplit()** | Разбивает строку справа | `'a,b,c'.rsplit(',', 1)` → `['a,b', 'c']` |
| **splitlines()** | Разбивает строку по переносам строк | `'line1\nline2'.splitlines()` → `['line1', 'line2']` |
| **join()** | Объединяет элементы итерируемого объекта в строку | `','.join(['a', 'b'])` → `'a,b'` |
| **replace()** | Заменяет все вхождения подстроки на новую строку | `'hello'.replace('l', 'L')` → `'heLLo'` |
| **zfill()** | Дополняет строку нулями слева до указанной длины | `'42'.zfill(5)` → `'00042'` |
| **ljust()** | Выравнивает строку по левому краю с заполнением | `'text'.ljust(10, '-')` → `'text------'` |
| **rjust()** | Выравнивает строку по правому краю с заполнением | `'text'.rjust(10, '-')` → `'------text'` |
| **expandtabs()** | Заменяет табуляции на пробелы (по умолчанию 8) | `'a\tb'.expandtabs(4)` → `'a   b'` |
| **isalnum()** | Проверяет, состоит ли строка только из букв и цифр | `'abc123'.isalnum()` → `True` |
| **isalpha()** | Проверяет, состоит ли строка только из букв | `'abc'.isalpha()` → `True` |
| **isdigit()** | Проверяет, состоит ли строка только из цифр | `'123'.isdigit()` → `True` |
| **isnumeric()** | Проверяет, являются ли все символы числовыми | `'½'.isnumeric()` → `True` |
| **isdecimal()** | Проверяет, являются ли все символы десятичными цифрами | `'123'.isdecimal()` → `True` |
| **isspace()** | Проверяет, состоит ли строка только из пробельных символов | `'   '.isspace()` → `True` |
| **islower()** | Проверяет, все ли символы в нижнем регистре | `'hello'.islower()` → `True` |
| **isupper()** | Проверяет, все ли символы в верхнем регистре | `'HELLO'.isupper()` → `True` |
| **istitle()** | Проверяет, начинается ли каждое слово с заглавной буквы | `'Hello World'.istitle()` → `True` |
| **encode()** | Кодирует строку в bytes с указанной кодировкой | `'привет'.encode('utf-8')` → `b'\xd0\xbf\xd1\x80...'` |
| **format()** | Форматирует строку, заменяя заполнители значениями | `'{} {}'.format('a', 'b')` → `'a b'` |
| **format_map()** | Форматирует строку с использованием словаря | `'{key}'.format_map({'key': 'val'})` → `'val'` |
| **casefold()** | Агрессивное приведение к нижнему регистру (для сравнений) | `'Straße'.casefold()` → `'strasse'` |
| **partition()** | Разделяет строку по первому вхождению разделителя | `'a-b-c'.partition('-')` → `('a', '-', 'b-c')` |
| **rpartition()** | Разделяет строку по последнему вхождению разделителя | `'a-b-c'.rpartition('-')` → `('a-b', '-', 'c')` |
| **maketrans()** | Создает таблицу перевода для translate() | `str.maketrans('ab', 'cd')` → `{97: 99, 98: 100}` |
| **translate()** | Заменяет символы согласно таблице перевода | `'abc'.translate({97: 'x'})` → `'xbc'` |
| **isascii()** | Проверяет, все ли символы ASCII (Python 3.7+) | `'hello'.isascii()` → `True` |
| **isidentifier()** | Проверяет, является ли строка допустимым идентификатором | `'var_name'.isidentifier()` → `True` |
| **isprintable()** | Проверяет, все ли символы доступны для печати | `'text\n'.isprintable()` → `False` |

> **Примечание**: Строки в Python неизменяемы, поэтому все методы возвращают новую строку.
