# Кортежи в Python

## Определение и создание
Кортеж (tuple) — это упорядоченная, неизменяемая коллекция элементов.

### Способы создания:
```python
# Пустой кортеж
tuple_1 = ()
tuple_2 = tuple()

# Кортеж с элементами
letters = ('a', 'b', 'c')
numbers = tuple([1, 2, 3])

# Кортеж из одного элемента (обязательна запятая)
single_element = ('a',)  # тип tuple
not_a_tuple = ('a')      # тип str
```

## Основные характеристики
- **Индексируются** (индекс начинается с 0)
- **Упорядочены** (сохраняют порядок элементов)
- **Неизменяемы** (после создания нельзя добавлять, удалять или изменять элементы)

### Пример доступа к элементам:
```python
letters = ('a', 'b', 'c')
print(letters[0])  # 'a'
```

## Неизменяемость
При попытке изменить элемент возникает ошибка:
```python
letters[0] = 'd'  # TypeError: 'tuple' object does not support item assignment
```

### Обходное решение:
```python
# Преобразование в список, изменение, обратное преобразование
letters = list(letters)
letters[0] = 'd'
letters = tuple(letters)
```

## Связь словарей и кортежей

### Просмотр элементов словаря
При использовании метода `.items()` для словаря в цикле с **одной переменной** получаем кортежи из пар "ключ-значение":

```python
shopping_dict = {'огурцы': 2, 'помидоры': 3, 'лук': 1, 'картофель': 2}

# Одна переменная — получаем кортежи
for item in shopping_dict.items():
    print(item)  # Каждый item — это кортеж
    print(type(item))  # <class 'tuple'>
```
**Вывод:**
```
('огурцы', 2)
<class 'tuple'>
('помидоры', 3)
<class 'tuple'>
('лук', 1)
<class 'tuple'>
('картофель', 2)
<class 'tuple'>
```

### Использование с `enumerate()`
Аналогичное поведение с функцией `enumerate()`:
```python
companies = ['Microsoft', 'Apple', 'Tesla']

# Одна переменная — получаем кортежи
for company in enumerate(companies):
    print(company, type(company))
```
**Вывод:**
```
(0, 'Microsoft') <class 'tuple'>
(1, 'Apple') <class 'tuple'>
(2, 'Tesla') <class 'tuple'>
```

## Распаковка кортежей
Распаковка позволяет присвоить элементы кортежа отдельным переменным:

### Базовая распаковка:
```python
a, b, c = ('a', 'b', 'c')
print(a)  # 'a'
```

### Распаковка в цикле с элементами словаря:
```python
shopping_dict = {'огурцы': 2, 'помидоры': 3, 'лук': 1}

# Две переменные — автоматическая распаковка кортежа
for key, value in shopping_dict.items():
    print(f"Продукт: {key}, Количество: {value}")
```
**Вывод:**
```
Продукт: огурцы, Количество: 2
Продукт: помидоры, Количество: 3
Продукт: лук, Количество: 1
```

### Распаковка с `enumerate()`:
```python
companies = ['Microsoft', 'Apple', 'Tesla']

# Две переменные — распаковка кортежа
for index, company in enumerate(companies):
    print(f"Индекс: {index}, Компания: {company}")
```

## Функция `zip()`
Объединяет элементы нескольких последовательностей в кортежи:

```python
names = ['Артем', 'Антон', 'Александр']
income = [97000, 110000, 95000]

# Создание zip-объекта (итератора из кортежей)
zipped = zip(names, income)

# Преобразование в список кортежей
result = list(zipped)
print(result)
# [('Артем', 97000), ('Антон', 110000), ('Александр', 95000)]
```

### Использование `zip()` в цикле:
```python
names = ['Артем', 'Антон', 'Александр']
income = [97000, 110000, 95000]

for name, salary in zip(names, income):
    print(f"{name}: {salary} руб.")
```

## Сводная таблица методов и операций с кортежами

| Метод/Операция | Описание | Пример |
|----------------|----------|--------|
| `count(x)` | Возвращает количество элементов со значением x | `t.count(5)` |
| `index(x)` | Возвращает индекс первого элемента со значением x | `t.index('a')` |
| `+` | Конкатенация (объединение) кортежей | `t1 + t2` |
| `*` | Повторение кортежа | `t * 3` |
| `in` | Проверка наличия элемента | `'a' in t` |
| `len()` | Возвращает количество элементов | `len(t)` |
| Индексация `[i]` | Доступ к элементу по индексу | `t[0]` |
| Срезы `[start:end]` | Получение части кортежа | `t[1:3]` |
| Распаковка | Присвоение элементов переменным | `a, b = t` |

## Ключевые выводы о связи словарей и кортежей:
1. Метод `.items()` возвращает объект представления, состоящий из кортежей (ключ, значение)
2. Функция `enumerate()` также возвращает кортежи (индекс, элемент)
3. Эти кортежи можно либо обрабатывать как единое целое, либо распаковывать на отдельные переменные
4. Такое поведение делает кортежи удобными для работы с парами данных

**Примечание:** Из-за неизменяемости у кортежей всего два метода (`count()` и `index()`), но они поддерживают все основные операции последовательностей.
