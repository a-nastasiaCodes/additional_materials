
## Что такое функция?
Функция — это **отдельный блок кода**, который:
- Выполняет конкретную задачу
- Может принимать данные (аргументы)
- Может возвращать результат
- Может использоваться много раз

## Как создать функцию?

```python
def название_функции():
    # что делает функция
    print("Привет!")
```

**Пример:**
```python
def сказать_привет():
    print("Привет, студент!")
```

## Как вызвать функцию?
Просто напишите её название со скобками:
```python
сказать_привет()  # Выведет: Привет, студент!
```

## Функция с параметрами

Параметры — это данные, которые мы передаём в функцию:

```python
def приветствие(имя):
    print(f"Привет, {имя}!")

приветствие("Анна")    # Привет, Анна!
приветствие("Максим")  # Привет, Максим!
```

## Функция, которая возвращает результат

```python
def сложить(a, b):
    результат = a + b
    return результат  # возвращаем результат

сумма = сложить(5, 3)
print(сумма)  # 8
```

Можно проще:
```python
def сложить(a, b):
    return a + b
```

## Параметры по умолчанию

Можно задать значение по умолчанию:

```python
def умножить(число, множитель=2):
    return число * множитель

умножить(5)      # 10 (5 * 2)
умножить(5, 3)   # 15 (5 * 3)
```

## Простой пример из жизни

```python
# Функция для вычисления площади прямоугольника
def площадь_прямоугольника(ширина, высота):
    return ширина * высота

# Используем несколько раз
площадь1 = площадь_прямоугольника(4, 5)  # 20
площадь2 = площадь_прямоугольника(2, 3)  # 6
```

## Коротко о главном

1. **Создать функцию:** `def название():`
2. **Вызвать функцию:** `название()`
3. **Передать данные:** `название(данные)`
4. **Вернуть результат:** `return результат`

## Пример для запоминания

```python
# 1. Создаём
def приготовить_чай(сахар=True):
    if сахар:
        return "Чай с сахаром"
    else:
        return "Чай без сахара"

# 2. Используем
мой_чай = приготовить_чай()  # с сахаром
твой_чай = приготовить_чай(False)  # без сахара
```

**Важно:** Сначала функция создаётся (def), потом используется!
